Case Study: exPose

PROJECT DETAILS

saniya$ cd expose



expose saniya$ position

>> Project Manager, iOS Engineer

expose saniya$ project-background

>> A 2-semester senior design project with two other teammates. Digital photography as a medium has become increasingly popular as camera technology continues to advance rapidly. Although the quality of the pictures taken have increased significantly, will the skills of the photographer be able to advance at the same rate? The best way to improve is not only through practice, but through critique. To accomplish this task, we created exPose, a IOS application that would be able to analyze the structure of a users photo using the Clarifai API and return not only useful metadata about the image, but also useful feedback to improve their photography.

expose saniya$ research

>> It was important to understand first: what modern solutions already exist in the market and how our application would be differ from them, how to implement our vision using different tools successfully in 3 sprints, and how to make our application user friendly. Research helped us find out that our application would differ in offering them insight on how they can improve their skills, when most photo improving applications only are really photo editing applications. We also chose to make an iOS application because it's the most popular for novice photographers - then we were able to find the ClarifaiAPI that can analyze photos and one can train models for further analysis. All these tools were new to us.
